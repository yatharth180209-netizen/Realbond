# RealBond

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yatharth180209-netizen/Realbond.git
   cd Realbond
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```

## Build Instructions

To build the project, run the following command:
```bash
npm run build
```
